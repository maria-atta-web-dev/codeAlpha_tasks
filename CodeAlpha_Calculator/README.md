## 🎯 Project Overview

This is **Task 2** of my CodeAlpha Frontend Development Internship - a modern, feature-packed calculator that goes beyond basic arithmetic. It's designed to impress with its beautiful UI, smooth animations, and powerful functionality.

---

## ✨ Features That Stand Out

### 🎨 **Visual Excellence**
- **Custom Cursor** - Interactive cursor that follows your mouse
- **Animated Background** - Floating gradient circles and particles
- **Dark/Light Theme** - Toggle between beautiful themes
- **Smooth Animations** - Button pops, ripples, and transitions
- **Glassmorphism Design** - Modern frosted glass effect

### 🧮 **Core Calculator Features**
- **Basic Operations** - Addition, subtraction, multiplication, division
- **Percentage Calculations** - Quick percent conversions
- **History Display** - Shows last calculation
- **Expression Preview** - See what you're calculating
- **Delete Function** - Backspace support

### 🔬 **Scientific Mode** (Bonus)
- **Trigonometry** - sin, cos, tan
- **Logarithms** - log, ln
- **Powers & Roots** - square, cube, sqrt, x^y
- **Constants** - π (pi), e
- **Factorial** - n! function

### ⌨️ **Keyboard Support** (Bonus)
- **Numbers** - 0-9 keys
- **Operators** - +, -, *, /
- **Enter/Equals** - Calculate
- **Escape** - Clear all
- **Backspace** - Delete
- **Decimal** - . key
- **Percent** - % key
