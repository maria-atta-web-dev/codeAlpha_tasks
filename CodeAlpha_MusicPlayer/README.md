# ✨ VIBE PLAYER — CodeAlpha Task 4

A stunning, feature-rich music player with a vibrant gradient design, smooth animations, and a curated playlist of popular hits. This project demonstrates advanced JavaScript audio control with a beautiful, responsive interface.

![Music Player Preview](preview.png) *(Add a screenshot here)*

## 🌟 Key Features

- **Vibrant Gradient Design**: Eye-catching purple-pink gradients with glass morphism
- **Animated Visualizer**: Dynamic sound bars that dance to the music
- **Curated Playlist**: 6 popular songs with album artwork from Unsplash
- **Smooth Controls**: Play, pause, next, previous with haptic feedback
- **Progress Seeking**: Click anywhere on the progress bar to jump
- **Volume Control**: Slider with percentage display and mute toggle
- **Keyboard Support**: Space (play/pause), ← (previous), → (next)
- **Active Track Highlight**: Current song glows in the playlist
- **3D Hover Effects**: Interactive elements with depth and shadow
- **Responsive Design**: Works perfectly on all devices

## 🚀 Quick Start

1. Create a folder named `CodeAlpha_MusicPlayer`
2. Inside, create these files:
   - `index.html` (copy the HTML code)
   - `style.css` (copy the CSS code)
   - `script.js` (copy the JavaScript code)
   - `README.md` (copy this documentation)
3. Open `index.html` in any modern browser
4. Click play and enjoy! 🎧
